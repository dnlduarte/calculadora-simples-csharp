# 🧮 Calculadora Simples em C#

Projeto básico desenvolvido em C# com o objetivo de praticar conceitos fundamentais de programação.

## 🚀 Funcionalidades

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 📊 Resto da divisão (módulo)
- ⚠️ Tratamento de divisão por zero

## 💻 Tecnologias utilizadas

![C#](https://img.shields.io/badge/C%23-Programming-blue)

## 📚 Conceitos praticados

- Entrada de dados com `Console.ReadLine()`
- Conversão de tipos (`int.Parse`)
- Operadores aritméticos
- Estruturas condicionais (`if/else`)
- Validação básica de erros
